# XPulse

XPulse: Twitter / X Video Downloader & Batch Zip Archive Exporter.

## Features
- **Single Video Mode**: Paste any tweet link to inspect author details and download pristine MP4 streams.
- **Batch Zip Exporter Mode**: Add multiple Twitter/X video links dynamically and download all videos compressed into an instant ZIP archive.

## Quick Start
```bash
npm install
npm start
```
Open `http://localhost:3000` in your web browser.
